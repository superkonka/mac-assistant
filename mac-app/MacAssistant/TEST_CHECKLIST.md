
## 🔍 如何提交问题

1. 打开日志查看器（菜单栏 → 📋 查看日志）
2. 点击 "🔍 诊断分析"
3. 点击 "📋 复制报告" 或 "💾 导出"
4. 将报告发送给我

## 🚀 重新启动应用

```bash
# 如果应用已运行，先退出
pkill MacAssistant

# 重新启动
open /Applications/MacAssistant.app
```

## 📝 已知限制

1. 首次运行可能需要授权：
   - 文件访问权限（保存日志）
   - 通知权限（系统提醒）

2. 需要安装 CLI 工具：
   - `kimi` (Kimi CLI)
   - `openclaw` (可选，系统分析)
