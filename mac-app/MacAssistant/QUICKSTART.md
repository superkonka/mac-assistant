# Mac Assistant v4.0.0 快速开始

## 日志系统

### 1. 查看日志

**方式一：状态栏菜单**
- 点击状态栏的 🧠 图标
- 选择 "📋 查看日志"（快捷键 ⌘+L）

**方式二：打开日志目录**
- 点击状态栏的 🧠 图标
- 选择 "📂 打开日志目录"

### 2. 日志功能

**日志查看器功能：**
- 📋 实时查看最近日志
- 🔍 关键词搜索
- 🎚️ 按级别过滤 (DEBUG/INFO/WARNING/ERROR/CRITICAL)
- ⏱️ 按时间范围筛选
- 🔄 自动刷新
- 💾 导出日志
- 🔍 **诊断分析** - 自动分析问题并生成报告

### 3. 诊断分析

1. 打开日志查看器
2. 点击 "🔍 诊断分析" 按钮
3. 查看诊断报告：
   - 问题分类和严重程度
   - 具体错误详情
   - 修复建议
4. 点击 "💾 导出" 保存报告

### 4. 日志文件位置

```
~/Documents/MacAssistant/Logs/
├── mac-assistant-current.log    # 当前日志
└── Archive/                     # 历史日志
```

### 5. 常见问题

**Q: 日志文件在哪里？**
A: `~/Documents/MacAssistant/Logs/mac-assistant-current.log`

**Q: 如何导出日志？**
A: 打开日志查看器 → 点击 "💾 导出"

**Q: 如何发送日志给开发者？**
A: 
1. 打开日志查看器
2. 点击 "🔍 诊断分析"
3. 点击 "📋 复制报告" 或 "💾 导出"
4. 将报告发送给开发者

**Q: 发现错误怎么办？**
A: 使用诊断分析功能，它会自动分析问题并给出修复建议。

### 6. 日志级别说明

| 级别 | 图标 | 说明 |
|------|------|------|
| DEBUG | 🔍 | 调试信息，最详细 |
| INFO | ℹ️ | 一般运行信息 |
| WARNING | ⚠️ | 警告，需要关注 |
| ERROR | ❌ | 错误，需要处理 |
| CRITICAL | 🚨 | 严重错误，必须处理 |

### 7. 开发中使用日志

```swift
// 记录调试信息
LogDebug("变量值: \(value)")

// 记录一般信息
LogInfo("操作完成")

// 记录警告
LogWarning("内存使用率超过80%")

// 记录错误
LogError("网络请求失败", error: networkError)

// 记录严重错误
LogCritical("数据库连接断开")
```
